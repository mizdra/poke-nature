# poke-nature-util
ポケモン性格ユーティリティクラス。

## 特徴
- TypeScriptによる型補完。

## インストール
```bash
$ npm install @mizdra/poke-nature-util
```

## ビルド
```bash
$ git clone https://github.com/mizdra/poke-nature-util.git
$ cd poke-nature
$ npm install -g  mocha gulp dtsm
$ npm install
$ dtsm install
$ gulp
```

### テスト
```bash
$ npm test
```
