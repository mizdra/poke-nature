# poke-nature
ポケモン性格クラス。

## 特徴
- TypeScriptによる型補完。

## インストール
```bash
$ npm install @mizdra/poke-nature
```

## ビルド
```bash
$ git clone https://github.com/mizdra/poke-nature.git
$ cd poke-nature
$ npm install -g  mocha gulp dtsm
$ npm install
$ dtsm install
$ gulp
```

### テスト
```bash
$ npm test
```
